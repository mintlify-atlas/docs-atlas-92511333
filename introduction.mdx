# Introduction

Parts Catalog is a comprehensive web application designed to manage and control a catalog of automotive parts sold by a company. Built with modern technologies, it provides a complete solution for cataloging parts, managing vehicles, and establishing relationships between them.

## What is Parts Catalog?

Parts Catalog is a full-stack CRUD application that allows you to:

- Create, list, update, and delete parts from your catalog
- Manage vehicle information
- Establish many-to-many relationships between parts and vehicles
- Search and filter parts by name or vehicle application
- Maintain detailed part specifications including weight information

## Architecture Overview

Parts Catalog is built with a modern, scalable architecture:

### Backend

- **Java 11** - Programming language
- **Spring Boot 2.4.3** - Application framework
- **Spring Web MVC** - REST Controller implementation
- **Spring Data JPA** - Database interaction layer
- **PostgreSQL** - Primary database for production
- **H2 Database** - In-memory database for unit testing
- **JUnit 5** - Unit testing framework
- **Maven** - Dependency management
- **Swagger** - API documentation and testing interface

### Frontend

- **Vue.js 2.6.11** - Progressive JavaScript framework
- **Vue Router 3.5.1** - Client-side routing
- **Axios 0.21.1** - HTTP client for API communication
- **Bootstrap** - Responsive UI components

### Database

The application uses PostgreSQL as its primary database with the following structure:

- **Parts Table** - Stores part information (name, application vehicle, weights)
- **Vehicles Table** - Stores vehicle information
- **Many-to-Many Relationship** - Links parts to their compatible vehicles

## API Capabilities

The REST API provides comprehensive endpoints for:

- **CRUD operations** for parts and vehicles
- **Search functionality** by part name or vehicle keyword
- **Relationship management** between parts and vehicles
- **Bulk operations** including delete all functionality
- **Alphabetical sorting** for organized data retrieval
