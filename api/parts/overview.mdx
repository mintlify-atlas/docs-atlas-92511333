## Introduction

The Parts API provides endpoints to manage automotive parts in the catalog system. You can create, retrieve, update, delete, and search for parts using this REST API.

## Base URL

```
http://localhost:8080/api/v1
```

## Authentication

Currently, the API does not require authentication. CORS is enabled for `http://localhost:8081` to allow frontend access.

## Common Response Codes

| Status Code | Description |
|-------------|-------------|
| 200 OK | Request successful |
| 201 Created | Resource created successfully |
| 204 No Content | Request successful, no content to return |
| 404 Not Found | Resource not found |
| 500 Internal Server Error | Server error occurred |

## Swagger Documentation

Interactive API documentation is available via Swagger UI:

```
http://localhost:8080/swagger-ui.html
```

## Part Object Schema

All endpoints use the following Part object structure:

<ResponseField name="id" type="long" required>
  Auto-generated unique identifier for the part
</ResponseField>

<ResponseField name="name" type="string" required>
  Name of the part
</ResponseField>

<ResponseField name="applicationVehicle" type="string">
  Vehicle application description
</ResponseField>

<ResponseField name="netWeight" type="double">
  Net weight of the part in the specified unit
</ResponseField>

<ResponseField name="grossWeight" type="double">
  Gross weight of the part in the specified unit
</ResponseField>

<ResponseField name="vehicles" type="Set<Vehicle>">
  Set of related vehicles (many-to-many relationship)
</ResponseField>

## Example Part Object

```json
{
  "id": 1,
  "name": "Polia",
  "applicationVehicle": "Fusca",
  "netWeight": 9.87,
  "grossWeight": 10.98,
  "vehicles": []
}
```

## Getting Started

To get started with the Parts API:

1. Ensure the backend is running at `http://localhost:8080`
2. Create your first part using the [Create Part](/api/parts/create) endpoint
3. Retrieve and list parts using the [Retrieve Parts](/api/parts/retrieve) endpoint
4. Use [Search](/api/parts/search) to find parts by name or vehicle application
